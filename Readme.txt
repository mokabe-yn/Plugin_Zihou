このソフトウェアは、みちあき氏が公開しているソフトウェア「棒読みちゃん」のプラグインです

機能
====

30分ごとに現在時刻を読み上げる。
読み上げ元のファイルは付属している `(プラグイン名)_Aloud.txt`。

1行目は"年"が変わったタイミング、
2行目は"月"が変わったタイミング、
3行目は"日"が変わったタイミング、
4行目は"時間"が変わったタイミング、
5行目は"分"が変わったタイミング、
6行目はそれ以外のタイミングで読み上げられる。

読み上げテキストは
`{0}`が年、
`{1}`が月、
`{2}`が日、
`{3}`が時間、
`{4}`が分、
`{5}`が秒 に展開される。

例: `{3}時{4}分` -> `22時45分`


注意点
------
- 読み上げ元のファイルはUTF-8。SJISだと読み込みエラーになる。

読み上げ話者の割り当て方
------------------------
読みちゃん側でのコマンドを利用する。
棒読みちゃんでは `y)`が「女性1」に割り当てられることを利用して、
`y){3}時です` のようにすれば「女性1」の声で時刻が読み上げられる。



開発者向け情報
==============

コンパイルに必要なもの
----------------------

- Visual Studio(バージョン問わず)
- .Net Framework 3.5開発環境
    - 今となっては化石の開発環境のため、VS2022からは明示的にインストールする必要がある
- BouyomiChan.exe本体
    - ここで定義されているinterfaceに乗っけるために必要

棒読みちゃんは再配布が禁止されているため、
このリポジトリに付属させていない。

`(このディレクトリ)/external/x64/BouyomiChan.exe` に手持ちの棒読みちゃんをコピーしてください

他のプラグインとの干渉
----------------------
このプラグイン自体が、棒読みちゃん付属のサンプルコードと干渉するため、同居できない。

サンプルコードを少しいじっただけのプラグインとも同じく同居できない。
